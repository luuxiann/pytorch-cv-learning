## 遇到的问题
git连接不上github仓库，无法上传文件。
![problem1](./pictures/problem1.png)
解决方法：尝试了网上的几个解决方法后，发现主要问题在于我的代理不对，一开始使用的代理是Watt Toolkit,也采用一下命令配置了git，仍无法连接。后面改用clash，并重新运行以下两行代码，成功连接上了github库，并成功上传了文件。
` git config --global http.proxy socks5://127.0.0.1:7890 `
` git config --global https.proxy socks5://127.0.0.1:7890 `
