# pytorch-cv-learning
## 总体学习目标
通过 3 个月（约 12 周）的项目驱动学习，掌握 Pytorch 基础语法与核心功能，熟悉计算机视觉常见任务（图像分类、目标检测、图像分割等）的流程与方法，具备独立完成小型计算机视觉项目的实践能力，并通过 GitHub 记录学习过程与成果，形成完整的学习日志。
### 文件介绍
按周分类，每周完成的任务放在一个week文件夹。
git_use.txt 和 markdown_use.txt 为git和markdown常用的命令语法。
## week1 (2025.11.01-2025.11.07)
初始先根据翁老师给的计划执行，实时根据自身情况调整
#### 学习目标
完成 Pytorch 的配置，熟悉 Pytorch 的使用
#### 学习任务
1. 配置好git和github仓库，使用git来上传和管理文件和代码。
1. 学习 Python 基础语法（变量、数据类型、控制流、函数等），预计 1 天。
1. 安装 Pytorch 及相关库（如 torchvision），了解其安装环境配置，预计 1 天。
1. 学习 Pytorch 张量（Tensor）的创建、操作（加减乘除、索引、切片等）、维度变换等知识，通过官方文档和简单示例练习，预计 2 天。

## week2 (2025.11.08-2025.11.14)
#### 学习目标
完成一个基于 Pytorch 的线性回归模型，能够对数据进行训练和预测
#### 学习任务
1. 学习 Pytorch 自动求导机制（Autograd），理解计算图原理，掌握如何通过张量的requires_grad属性实现自动求导，预计 2 天。
2. 学习 Pytorch 神经网络模块（nn.Module）的使用，搭建简单的线性回归模型并进行训练，预计 2 天。
